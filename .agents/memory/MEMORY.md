- [Vite API Proxy Required](vite-proxy.md) — Without a Vite proxy, browser fetches to /api/* hit the SPA fallback and return HTML strings, causing .map errors.
- [Lib Declarations Must Be Built First](lib-build-order.md) — api-client-react and api-zod use composite TS project references; run `pnpm run typecheck:libs` before typechecking any artifact or errors about "Output file has not been built" appear.
- [Auth Architecture](auth-architecture.md) — JWT auth via bcryptjs + jsonwebtoken; users + orders tables in DB; AuthContext in frontend; checkout shows auth banner but allows form fill, gates submission behind login modal.
- [Seed Run Command](seed-run.md) — No npm seed script; run with `npx tsx lib/db/src/seed.ts` from workspace root. Seed covers categories, chefs, menu items, testimonials, blog posts, subscription plans.
- [Paymob Integration Pattern](paymob-integration.md) — Full 3-step Paymob flow (auth token → order → payment key → iframe URL) in `artifacts/api-server/src/lib/paymob.ts`; gated on env vars PAYMOB_API_KEY / PAYMOB_INTEGRATION_ID / PAYMOB_IFRAME_ID; checkout shows Paymob iframe inline when card selected and keys are set.
